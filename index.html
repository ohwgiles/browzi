Browzi

Chinese character (hanzi) browser based on the Unihan database

Download this project as a .zip file Download this project as a tar.gz file

Browzi is a graphical browser for looking up detailed information on Chinese characters (汉字). It is based on the Unihan Database via libunihan

Browzi is inspired by the GPL project Plain Hanzi Editor and uses some of its code.
